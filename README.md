# Проект "Оно тебе надо"

Аукцион вещей, в которые никто не верил.

## О проекте
Это учебный проект Яндекс Практикума. Верстка выполнена по макету в Figma.

## Технологии
- HTML5
- CSS3
- Flexbox
- Grid Layout

## Структура проекта
- `index.html` — главная страница
- `styles/style.css` — основные стили проекта
- `styles/global.css` — глобальные стили (сброс браузерных настроек)
- `fonts/` — папка со шрифтами
- `images/` — папка с изображениями
